Navyanta is a premium aluminium fabrication service platform built on the **MERN stack** (MongoDB, Express, React, Node.js).  
It provides a seamless experience for customers to explore certified fabrication services and for admins to manage workflows, quotations, and notifications.

  - Modular routers for admin, user, service, and category
  - Notification system (SMS via MSG91, WhatsApp via Gupshup, Email via AWS SES)
  - Secure authentication and role-based access
